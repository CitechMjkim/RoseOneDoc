      디바이스 체크 서버 인터벌 훅훅

      Returns { checkServer: () => { start: (ip: string) => void; stop: () => void } }

      디바이스 서버 체크 훅

      Example

      const { checkServer } = useDeviceCheckServer();
      const checkServerState = checkServer();

      checkServerState.start("192.168.0.1");
      checkServerState.stop();

      Remarks

      • 해당 훅은 앱 실행 후 연결된 세트에 체크 서버를 보내는 인터벌 훅입니다.
      • 체크 서버는 5초마다 보내며, 성공: 200, 한번 실패: 500, 연속 실패: 404 코드를 useCheckServer Context에 저장합니다. 저장된 Context는 Header에 리모콘 아이콘에 사용됩니다.
      • 체크 서버 API 는 postDeviceCheckServer 함수를 사용하며 리턴값은 IDeviceCheckServer 타입입니다.
