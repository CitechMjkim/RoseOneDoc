      로컬 네트워크 상에서 디바이스 또는 앰프를 탐색하기 위한 커스텀 훅

        서비스 검색 옵션

        • ip: string

          count: number;

      }

      Remarks

      • Zeroconf(Bonjour)를 사용하여 네트워크 상의 서비스들을 검색합니다.
      • type이 "device"일 경우: roseHifi- 네이밍을 포함하며 AMP를 제외한 디바이스만 필터링합니다.
      • type이 "amp"일 경우: AMP 키워드가 포함된 서비스만 필터링합니다.
      • .local 도메인 스캔과 함께 /deviceName API 호출을 통해 장치 정보를 보완 수집합니다.
      • 병렬 네트워크 스캐닝은 최대 70개씩 묶어서 처리되며, 중복은 제거됩니다.

      Example

      const { serviceDiscovery, serviceList, count } = useServiceDiscovery({ type: "device", ip: "192.168.1.10" });

      useEffect(() => {
        serviceDiscovery.start();
        return () => {
          serviceDiscovery.stop();
        };
      }, []);

      console.log("총 검색된 기기 수:", count);
      console.log("기기 리스트:", serviceList);
